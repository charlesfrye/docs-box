#### Examples:

Run a sample sweep over a function:
```
def train():
    with wandb.init() as run:
        print("config:", dict(run.config))
        for epoch in range(35):
            print("running", epoch)
            wandb.log({"metric": run.config.param1, "epoch": epoch})
            time.sleep(1)

wandb.agent(sweep_id, function=train)
```
